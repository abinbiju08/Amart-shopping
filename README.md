# A-Mart Shopping

A full-featured e-commerce web application built with React, Vite, and React Router.

## Features
- 🛍️ Product listing with search and quantity selection
- 🛒 Cart management with live badge updates
- 💳 Checkout with delivery options and live price calculation
- 📦 Order history with buy again functionality
- 🚚 Package tracking with progress bar

## Tech Stack
- React 
- React Router
- Vite
- CSS

## Pages
- **Home** — Browse and search products, add to cart
- **Checkout** — Review cart, choose delivery, place order
- **Orders** — View order history, buy again
- **Tracking** — Track package delivery status

## Getting Started
```bash
npm install
npm run dev
```
